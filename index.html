About JPHL

JPHL (Java Paging Hyper Language) is designed to facilitate website creation using most of the HTML syntax, along with custom tags for extended functionalities. This language allows for building modern websites that run on JavaScript servers.

Documentation

JPHL uses 80% of the standard HTML syntax while adding custom tags for additional functionalities. Here's a basic structure of a JPHL file:


                
                    
                        Sample JPHL Page
                    
                    
                        

                            
Welcome to JPHL!

                        

                        

                            
This is a simple JPHL page.

                        

                        

                            
Created using JPHL

Installation Instructions

Follow these steps to install and run the JPHL server:

  1. Download the JPHL server executable.
  2. Place the `.exe` file in your project directory.
  3. Open your terminal or command prompt.
  4. Run the server with the command:
  5. https://github.com/Carti-drillman/JPHL-language/releases/tag/server
node JP_server.js your-file.jphl

Ensure Node.js is installed and added to your system's PATH.
